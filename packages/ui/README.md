# UI Package

Reusable UI components and utilities.

## Install & Use
```ts
import { Button } from '@kdx-kit/ui'

export default function Page() {
  return <Button>Click me</Button>
}
```

## Development
- `pnpm run dev` in `packages/ui`
- `pnpm run build` to produce output

## Diagnostics
- Lint/type-check via Turbo pipeline from repo root.
- See root README for diagnostics and scripts.
